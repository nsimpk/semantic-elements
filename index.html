HTML5 Semantic Elements

What are Semantic Elements?

In the world of HTML a semantic element is an element that clearly defines its content. A non-semantic element wouldn't define it's content whatsoever. While the code itself may perform the same function whether it uses semantic elements or not, reading the code is much more difficult. In conclusion, the use of semantic elements clearly describes its meaning to both the browser and the developer.
